## README

Per the [GitHub ROB][gh-rob] and [NIST Suborder 1801.02][nist-s-1801-02],
your README should contain:

1. Software or Data description
   - The purpose of this data is for use in reproducing molecular dynamics simulations published in the article "Assembly of a DNA tile electron spin breadboard" by Veronika A. Szalai, Christina Bergonzo, and Jacob Majikes.
   - The repository contains Amber formatted lib and frcmod files for the EPR spin label TEMPO tag, incorporated into the DNA tile.
   - Technical instructions to compile Amber are found elsewhere. File format information for Amber frcmod files can be found here: https://ambermd.org/FileFormats.php#frcmod
